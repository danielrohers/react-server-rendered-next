# react-server-side-next

Sample application using [Next.js](https://github.com/zeit/next.js), a minimalistic framework for server-rendered React applications.

## Install

```bash
[sudo] npm install
```

or

```bash
yarn
```

## Development

Start
```bash
npm run dev
```

[ESLint](http://eslint.org)
```bash
npm run eslint
```

## Deploy
```bash
npm run build && npm start
```
or
```bash
npm run deploy
```
